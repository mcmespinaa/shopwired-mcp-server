One powerful integration

Everything you need — from checkout to catalog, all the essentials are built in.

Every tool declares MCP annotations (read-only, destructive, idempotent) so your AI client knows what's safe to auto-approve — and every delete demands explicit confirmation.

Get started

Two minutes from zip to first prompt.

1

Get the code

Download the zip above, or clone from GitHub.

git clone https://github.com/
  mcmespinaa/shopwired-mcp-server.git
2

Run the installer

Sets up Python, asks for your ShopWired API key & secret, registers with Claude Code.

cd shopwired-mcp-server
bash install.sh  # Windows: install.bat
3

Ask your store something

Restart Claude and try:

"How many orders came in
 this week?"
