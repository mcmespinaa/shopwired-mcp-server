One powerful integration

Everything on the shelf, nothing off the books.

Every tool declares MCP annotations (read-only, destructive, idempotent) so your AI client knows what's safe to auto-approve — and every delete demands explicit confirmation.

🛍️11 tools

Products

Search, list, create and update — stock levels, variations, images. Pence-to-pounds handled correctly.

📦7 tools

Orders

List, search, update status with optional customer notification, add internal comments.

💛4 tools

Customers

Look up, count and create customer records — with email validation built in.

🗂️16 tools

Store configuration

Categories, brands, vouchers, gift cards, shipping zones, webhooks (HTTPS-only), payment methods.

🛡️built in

Reliability layer

Leaky-bucket rate limiter matched to ShopWired's API limits, circuit breaker, response cache, retry with backoff.

☁️optional

Remote mode

Deploy as a hosted MCP server: bearer-token auth, /health probes, Dockerfile for Cloud Run, Render or any VPS.

Built-in safety, by default

Deletes require confirm=True Read-only tools auto-approvable Input validation on every tool Sanitized error messages HTTPS-only webhooks Credentials never logged
Get started

Two minutes from zip to first prompt.

1

Get the code

Download the zip above, or clone from GitHub.

git clone https://github.com/
  mcmespinaa/shopwired-mcp-server.git
2

Run the installer

Sets up Python, asks for your ShopWired API key & secret, registers with Claude Code.

cd shopwired-mcp-server
bash install.sh  # Windows: install.bat
3

Ask your store something

Restart Claude and try:

"How many orders came in
 this week?"
